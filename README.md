# fortunatesonpeak
This is a mod i build for the video game PEAK.

Its a dumb mod made with c# that at the end of the game, adds a song when the helicopter arrives to rescue everyone.

It was fun making it and more fun seeing it being downloaded by almost 16.000 people on thunderstore (a mod manager) https://thunderstore.io/c/peak/p/Aesthetic45/FortunateSonEndGameSong/
